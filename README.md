# homepage
StatHack の会社ホームページ

## Usage
### Setup
```
$ cd homepage
$ sh docker/build.sh
```

### Start the server
```
$ sh docker/run.sh
```

### Run shell in the docker container
```
$ sh docker/exec.sh
```
